Políticas de Privacidade

Bem-vindo à Política de Privacidade da RotusQuando o usuário usa nossos serviços, confia a nós informações dele. Esta Política de Privacidade destina-se a ajudar o usuário a entender quais dados coletamos, por que os coletamos e o que fazemos com eles. Como sabemos que isso é importante, esperamos que o usuário tire algum tempo para lê-la com cuidado. Lembramos que é possível encontrar os controles para gerenciar informações e proteger a privacidade e segurança em Minha conta.

Dados Pessoais que Coletamos
A Rotus coleta dados para atuar de forma eficaz e lhe proporcionar uma melhor navegação em nosso sistema. Alguns desses dados são fornecidos diretamente, por exemplo, quando você cria uma conta. Todos os dados coletados são para uso da instituição da qual a pessoa faz parte, sendo que nós só coletamos os dados para o uso exclusivo nessa instituição. Para excluir estes dados você devera se informar com a instituição da qual você participa para eles entrarem em contato conosco.

Como Acessar e Controlar seus Dados Pessoais
Você pode acessar seus dados pessoais diretamente pelo aplicativo, mas depois de ter cadastrado suas informações pessoais não será possível modifica-las.

Segurança de Dados Pessoais
A Rotus está empenhada em proteger a segurança de seus dados pessoais. Utilizamos uma vasta gama de tecnologias e procedimentos de segurança para ajudar a proteger seus dados pessoais contra acesso, utilização ou divulgação não autorizados. Por exemplo, armazenamos seus dados pessoais fornecidos em sistemas informáticos com acesso limitado e localizados em instalações controladas.

Onde Armazenamos e Processamos Dados Pessoais
Os dados pessoais coletados pela Rotus podem ser armazenados e processados no Brasil, Normalmente, o local de armazenamento principal é na região do cliente, muitas vezes com um backup de data center em outra região. Os locais de armazenamento são escolhidos para operar com eficiência, melhorar o desempenho e criar redundâncias para proteger os dados no caso de uma interrupção ou outros problemas. Nós tomamos as medidas necessárias para garantir que os dados que coletamos segundo esta política de privacidade sejam processados de acordo com as disposições desta instrução e com os requisitos da lei aplicável do local onde os dados se encontram.

Vamos atualizar esta política de privacidade para refletir os comentários dos clientes e as alterações aos nossos produtos. Quando publicarmos alterações nesta política, atualizaremos a "data de atualização" informando no site.
